Step 1: Create an Azure free account

Go to portal.azure.com

Sign in with your Microsoft account

If you are a student → choose Azure for Students


Step 2: Create a Virtual Machine

In Azure portal, search Virtual Machines

Click Create → Azure virtual machine

Step 3: Fill basic details 

On the Basics page:

Subscription: Free / Student subscription

Resource group: Click Create new → name rm1

Virtual machine name: vm1

Region: Any (India / nearby if available)

Image: Ubuntu 20.04 or 22.04

Size: B1s (this is free / cheapest) ✅

Username: azureuser

Authentication: SSH

Step 4: Connect to the VM

After VM is created:

Copy Public IP address
ssh azureuser@PUBLIC_IP
